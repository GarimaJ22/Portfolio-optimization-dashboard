# 📊 Portfolio Optimization Dashboard

This project demonstrates portfolio optimization using Modern Portfolio Theory and an interactive dashboard built with Dash.

## 🚀 Features
- Portfolio allocation optimization (Sharpe Ratio)
- Efficient frontier visualization
- Stock price trends
- Correlation heatmap

## 🛠️ Tech Stack
- Python
- Pandas, NumPy
- Plotly, Dash
- yFinance

## 📈 Results
- Optimized portfolio based on risk-return tradeoff
- Interactive dashboard for analysis

## ▶️ How to Run
1. Install dependencies: pip install dash yfinance numpy pandas scipy plotly
2. Run: python app.py
3. Open: http://127.0.0.1:8050/

- Portfolio shows strong performance
