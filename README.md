# 📊 Portfolio Optimization Dashboard

This project demonstrates portfolio optimization using Modern Portfolio Theory and an interactive dashboard built with Dash.

## 🚀 Features
- Portfolio allocation optimization (Sharpe Ratio)
- Efficient frontier visualization
- Stock price trends
- Correlation heatmap

## 🛠️ Tech Stack
- Python
- Pandas, NumPy
- Plotly, Dash
- yFinance

## 📈 Results
- Optimized portfolio based on risk-return tradeoff
- Interactive dashboard for analysis

## ▶️ How to Run
1. Install dependencies: pip install dash yfinance numpy pandas scipy plotly
2. Run: python app.py
3. Open: http://127.0.0.1:8050/

## Key Insights

- The optimized portfolio achieves a strong balance between risk and return.

- A Sharpe Ratio greater than 1 indicates good risk-adjusted performance.

- The model allocates higher weights to stocks with better return-to-risk efficiency.

- Some assets may receive 0% allocation, indicating they do not improve the portfolio.

- Diversification benefits are observed through the correlation heatmap.

## 📈 Project Objective
To apply quantitative finance techniques and build an interactive tool that helps in making data-driven investment decisions.

## 📌 Conclusion
This project highlights how data science and finance can be combined to construct optimal portfolios and visualize investment strategies effectively
